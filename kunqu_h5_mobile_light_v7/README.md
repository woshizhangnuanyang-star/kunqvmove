# 昆曲韵动 H5 WebAPP Demo - 移动端轻量优化版 v7

本版本是为手机扫码访问做的轻量优化包，重点解决手机端打开超时、白屏、加载过慢的问题。

## 已优化
- APP 页面、长页面、底部导航素材已转换为 WebP。
- 电视端大图已从原 3840×2400 PNG 压缩为轻量 WebP。
- 删除重复的 `assets/app` 原图目录，只保留实际运行目录 `assets/app_content`。
- 关闭 Service Worker 缓存注册，避免手机端缓存旧资源导致打不开。
- 保留原有交互：顶部时间栏固定、底部导航固定、页面内容单独滚动、连接大屏直达电视端、退出游戏返回 APP 首页。

## 部署方式
1. 解压这个文件夹。
2. 进入文件夹，确认第一层能看到：`index.html`、`app.js`、`styles.css`、`assets`、`tv`。
3. 在 Netlify 里选择手动部署，把这些文件整体拖进去。不要只上传压缩包。
4. 部署完成后，手机端建议先用 Safari / Chrome / Edge 打开链接测试。
5. 如果是覆盖旧站点，链接末尾加 `?v=7`，例如：`https://你的地址.netlify.app/?v=7`。
