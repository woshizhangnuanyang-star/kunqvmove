# 昆曲韵动 H5 WebAPP Demo v7

本版本基于 v6 优化：

1. 电视端资源压缩：将 3840×2400 的 TV 端 PNG 背景与动作帧压缩为 1920×1200 WebP，显著减少首屏资源体积。
2. 电视端移动适配：手机端/小屏端缩小摄像头卡片、评分卡片、曲目热区等视觉元素，避免在移动端显得过大。
3. 体感引擎启动优化：页面不再被 TensorFlow 外链脚本阻塞；模型脚本后台预热，开始训练时优先加载姿态识别，切换曲目时再加载手势识别。
4. 流畅度优化：摄像头分辨率和 Canvas DPR 按设备能力自动降低，移动端检测频率更稳。
5. 保留 v6 的交互：手机端 APP、固定顶部/底部导航、连接大屏直达电视端、退出游戏返回首页。

## 部署方式

Netlify / Vercel 部署时，进入解压后的文件夹，上传这一层中的所有内容：

```text
index.html
app.js
styles.css
assets/
tv/
manifest.webmanifest
sw.js
netlify.toml
```

不要只上传压缩包，也不要多包一层外层文件夹。

## 手机端打不开的情况

如果手机端显示 `ERR_CONNECTION_TIMED_OUT`，通常是手机网络访问 Netlify/Vercel 节点不稳定，不一定是代码问题。可尝试：

- 复制链接到手机浏览器打开，不只用微信内置浏览器；
- 切换手机流量 / Wi-Fi / 电脑热点；
- 同时准备 Netlify 和 Vercel 两个备用链接；
- 比赛现场网络不稳定时，可用电脑热点或本地服务器做备用演示。
