# 昆曲韵动 H5 高保真体验 Demo（v4）

本版本根据最新反馈，重点替换了长页面的固定底部导航素材，并重新校对了导航热区位置。

## 本次修改重点

- 首页长页面底部导航已替换为 `navigation-home-long.png`，对应首页选中状态。
- 记录类长页面底部导航已统一替换为 `navigation-record-long.png`，对应记录页选中状态。
- 底部导航高度按新素材重新适配：长页面滚动时只滚动页面内容，导航栏固定在手机底部。
- 底部导航点击热区重新校准为五等分逻辑： 首页 / 设备 / 连接大屏 / 记录 / 我的，对应新素材位置。
- 继续保留 v3 的移动端适配、电视端直连、退出游戏返回 APP 首页等交互逻辑。

## 核心体验路径

1. 启动页 → 引导页 → 调查页 → 登录/注册 → 连接设备 → 首页
2. 首页 → 进入商城 → 皮肤商城
3. 首页 → 查看曲库 → 曲库 → 动作库 → 电视端体感页面
4. 首页 → 你的计划 → 训练计划 → 电视端体感页面
5. 底部导航 → 记录 → 日/周/月记录 → 查看成就 → 徽章页面
6. 底部导航 → 我的 → 查看成就 → 徽章页面
7. 底部导航 → 连接大屏 → 电视端页面 → 切换曲目 → 退出游戏 → 返回 APP 首页

## 部署方式

直接将整个文件夹拖到 Netlify / Vercel / 静态服务器即可。部署成功后，把生成的网址制作成二维码，评委扫码即可体验。
